🌸 Kotoba — Learn Japanese Smarter

Kotoba is an AI-powered Japanese language learning platform built using Next.js, Clerk Authentication, and language APIs.

The platform helps users learn Japanese through:

* Hiragana & Katakana learning
* AI Translation
* Flashcards
* Quiz system
* Japanese sentence learning
* Beginner learning modules
* User authentication and profile system
